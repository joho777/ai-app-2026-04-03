# AgentOps Canvas

A single-file, interactive web app inspired by **Qwen3.6-Plus: Towards real world agents**.

## What it is
AgentOps Canvas is a visual playground for designing and simulating tool-using AI agents:
- Build an agent graph (Planner, Researcher, Coder, Critic, Tool nodes)
- Configure policies (budget, risk tolerance, stop conditions)
- Run a deterministic simulation of multi-step tool use
- Inspect traces, token/cost estimates, failure modes, and guardrails
- Compare strategies (ReAct vs Plan-Execute vs Reflexion) side-by-side

## Run locally
Open `index.html` in any modern browser.
